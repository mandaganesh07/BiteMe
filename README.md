# Biteme - Food Delivery & Quick Commerce Platform

🚀 **A modern, scalable food delivery and quick commerce application** built with cutting-edge technologies.

## 📋 Project Overview

Biteme is a comprehensive platform that combines:
- **Food Delivery** (restaurant orders with real-time tracking)
- **Quick Commerce** (grocery & essentials delivery in minutes)
- **Admin Dashboard** (comprehensive management system)
- **Multi-platform Support** (Web, iOS, Android)

## 🏗️ Architecture

```
Biteme/
├── backend/              # NestJS API Server
├── frontend/             # React Web Application
├── mobile/              # React Native Mobile App
├── admin-panel/         # React Admin Dashboard
├── docs/                # Documentation
└── docker-compose.yml   # Docker orchestration
```

## 🛠️ Tech Stack

### Backend
- **Framework**: NestJS (TypeScript)
- **Database**: PostgreSQL + Redis
- **Real-time**: Socket.io
- **Authentication**: JWT + OAuth2
- **Payments**: Stripe integration
- **File Upload**: AWS S3 / MinIO

### Frontend (Web)
- **Framework**: React 18 + TypeScript
- **State Management**: Redux Toolkit
- **UI Library**: Tailwind CSS
- **Maps**: Google Maps API

### Mobile (iOS/Android)
- **Framework**: React Native + TypeScript
- **Maps**: React Native Maps
- **Push Notifications**: Firebase

### Admin Panel
- **Framework**: React 18 + TypeScript
- **UI**: Ant Design
- **Charts**: Recharts

## 🎯 Key Features

✅ User authentication & profiles
✅ Browse restaurants & food items
✅ Real-time order tracking with GPS
✅ Multiple payment methods

✅ Admin management system

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 14+
- Redis 7+

### Installation

```bash
git clone https://github.com/mandaganesh07/Biteme.git
cd Biteme
docker-compose up -d
```

## 📚 Services

- **Backend API**: http://localhost:3000
- **API Docs**: http://localhost:3000/api/docs
- **Frontend**: http://localhost:3000
- **Admin Panel**: http://localhost:3001
- **Database UI**: http://localhost:8080
